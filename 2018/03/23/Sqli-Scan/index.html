SQLI SCAN

Sqli_Scan , 批量sql注入扫描器

首先从浏览器（Bing、Google、Yahoo）上根据关键词搜索符合条件的链接，再根据链接进行自动判断是否可能存在注入

依赖

  • python3

下载

基本使用

执行 python sqli_scan.py -k inurl:php?id= -p 3 -e bing
