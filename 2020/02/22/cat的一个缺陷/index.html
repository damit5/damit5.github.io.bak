cat的一个缺陷

2020-02-22 PV:

说明

  • cat有一个缺陷，如果内容中存在 \r 回车符，那么在cat直接查看的时候会首先输出\r后的内容，如果\r后的内容长度小于 \r 前的内容长度，那么就会输出 \r 后的内容在加上 \r 前的长度差的后几位。

  • 利用这个可以隐藏在一些脚本里面，比如 crontab， 直接 cat file 的查看方式会让运维不能发现问题。但是可以通过 vim less 查看到。
